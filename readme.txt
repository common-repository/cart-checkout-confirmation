Cart Checkout Confirmation plugin will give you a step to confirm information to proceed to checkout

== Description ==

Cart Checkout Confirmation is a plugin that allows you to insert a confirmation popup between WooCommerce’s checkout screen (the page where you enter the customer information) and the checkout completion screen.

It also has a [pro](https://wp.and-bro.com/shop/plugin/cart-checkout-confirmation) version to upgrade your plugin.
Enabling you to
・choose between a confirmation popup or a page
・change and reorder the items on the confirmation page by simple drag and drops
・change the color of the confirmation button
・change the confirmation button’s text
・put a text in the confirmation page between the Purchaser Information and the Cart Information

== Installation ==

1. Cart Checkout Confirmation can be installed directly through your WordPress
Plugins dashboard.
2. Click "Add New" and Search for "Cart Checkout Confirmation"
3. Install and Activate
